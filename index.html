🌐 TorClient Demo

What this demonstrates:

This demo shows advanced TorClient features including:

  • 🔄 Circuit Reuse: Persistent circuits that are reused across multiple requests
  • ⏰ Auto-Updates: Circuits automatically refresh every 2 minutes with graceful transition
  • 🌐 Individual Requests: Make requests to custom URLs using the persistent circuit
  • 🔒 Isolated Requests: Temporary circuits for maximum privacy on single requests
  • 📊 Live Status: Real-time circuit status and update information
  • ⚡ Manual Updates: Force circuit refresh with custom deadlines
⚠️ Note: Initial circuit creation can take 30-90 seconds. The demo uses 2-minute auto-updates for demonstration - normally this would be 10 minutes.
